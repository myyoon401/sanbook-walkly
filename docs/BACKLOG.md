# 📋 Walkly 개발 백로그

> Kotlin 학습을 위한 걸음수 랭킹 / 산책 도우미 앱 백엔드
> 작성일: 2026-04-20

---

## 🏃 Phase 1. MVP — "걸음수 기록하고 순위 보기"

목표: 2주 안에 "내 걸음수를 저장하고 랭킹을 볼 수 있다"까지 완성

| # | 기능 | 학습 포인트 (Kotlin) |
|---|------|---------------------|
| 1 | **User 도메인 분리** — Walker → User 로 리네이밍, 이메일/닉네임/프로필 | `data class`, nullable 타입 |
| 2 | **회원가입 / 로그인 API** — JWT 또는 세션 기반 | sealed class로 인증 결과 표현 |
| 3 | **걸음수 기록 API** — `POST /steps` (날짜별로 누적/덮어쓰기) | `LocalDate`, 확장 함수 |
| 4 | **일일 걸음수 조회** — `GET /steps?date=2026-04-20` | 기본 파라미터, `?:` 엘비스 연산자 |
| 5 | **전체 랭킹 조회** — 오늘/이번주/이번달 기준 Top N | `when` 표현식, `groupBy`/`sumOf` 컬렉션 함수 |

## 🗺️ Phase 2. 산책 경험 — "산책을 기록하자"

| # | 기능 | 학습 포인트 |
|---|------|-------------|
| 6 | **산책 세션 시작/종료** — `POST /walks/start`, `POST /walks/{id}/end` | enum class (상태: IN_PROGRESS/DONE) |
| 7 | **경로(위도/경도) 기록** — 산책 중 좌표 배치 업로드 | data class 중첩, list 처리 |
| 8 | **산책 통계** — 거리, 시간, 평균 속도 계산 | 확장 프로퍼티로 거리 계산 |
| 9 | **산책 히스토리 목록** — 내 지난 산책 목록 (페이징) | Pageable, 람다 |
